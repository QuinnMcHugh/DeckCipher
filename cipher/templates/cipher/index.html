Generate key

Enciphering a plaintext message requires you to have a key. That same key is also necessary when deciphering an encoded message. To obtain a random deck ordering, use the generate button below. This produces 52 randomly ordered cards represented in text ("6s" means Six of Spades). deck ordering is the only thing keeping your message secure, so keep it secret!

Encipher/Decipher message

Once you have generated a key, write a message which you want to securely transmit. The message input goes in the left text-box below, and your required key goes in the Key field. The blue arrow converts the message into ciphertext which can then be securely transmitted. When receiving an encrypted message, put the ciphertext in the left text-box and change the Action field to Decipher.
